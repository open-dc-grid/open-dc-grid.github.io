# Grid Standard

This standard is developed collaboratively in the GitHub repository open-dc-grid/standard (opens new window).

The website open-dc-grid.org/standard (opens new window) is only updated after each release. If you want to participate in the development, post issues (opens new window) or send pull-requests (opens new window) on GitHub.

It is planned to auto-generate PDF documents from the markdown source files in the future.
